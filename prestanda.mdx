## Prestandaöversikt

Översikten visar poäng för viktiga områden.

<AccordionGroup>

## Core Web Vitals

Core Web Vitals visar hur snabbt och stabilt sidan upplevs.

<AccordionGroup>
  <Accordion title="First Contentful Paint (FCP)">
    Hur snabbt det första innehållet visas.
  </Accordion>

  <Accordion title="Largest Contentful Paint (LCP)">
    Hur snabbt det största synliga innehållet laddas.
  </Accordion>

  <Accordion title="Total Blocking Time (TBT)">
    Hur länge sidan blockeras innan den kan reagera på användarens input.
  </Accordion>

  <Accordion title="Cumulative Layout Shift (CLS)">
    Hur mycket layouten hoppar medan sidan laddas.
  </Accordion>

  <Accordion title="Speed Index">
    Hur snabbt sidans innehåll visuellt byggs upp.
  </Accordion>
</AccordionGroup>

## Prestanda över tid

Du kan följa hur prestandapoängen utvecklas över tid.

Periodväljaren kan visa exempelvis senaste 7, 30, 90 eller 180 dagarna samt senaste året.

## Koldioxidavtryck

Om tillräcklig data finns kan sidan visa en uppskattning av webbplatsens koldioxidavtryck per sidvisning.

<Note>
  Koldioxidavtrycket är en uppskattning baserad på sidans storlek och bör användas som vägledning, inte som exakt miljörapport.
</Note>

## Bildanalys
