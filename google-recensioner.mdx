title: "Google-recensioner"
description: "Fält och inställningar för blocket Google-recensioner."
---

Blocket **Google-recensioner** visar recensioner och betyg från Google.

## Fält

### Titel och text

Rubrik och introduktion för blocket.

### Google Place ID

Place ID används för att hämta recensioner från rätt plats på Google.

### Minst antal stjärnor

Filtrerar vilka recensioner som visas.

Du kan till exempel visa alla recensioner eller endast recensioner från 4 stjärnor och uppåt.

### Antal recensioner att visa

Styr hur många recensioner som visas.

### Visa sammanfattning

Visar sammanfattning med betyg och antal recensioner från Google.

### Kolumner och layout

Styr hur recensionerna visas, till exempel i grid eller karusell.

## Designinställningar

Du kan även styra titelstorlek, textstorlek, rubriknivå, tema, vertikal padding, ramar och ankar-ID.

<Note>
  Blocket kräver ett korrekt Google Place ID för att visa recensioner.
</Note>
