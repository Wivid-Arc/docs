Inläggets block är de innehållsdelar som bygger upp en artikel.

Blocken är enklare och mer fokuserade än sidornas fulla blockbibliotek. De är gjorda för löpande innehåll som text, bilder, fördelar, frågor och formulär.

## Lägga till block

1. Öppna ett inlägg i redigeraren.
2. Gå till **Inläggets innehåll** i vänster panel.
3. Klicka på **Lägg till block**.
4. Välj blocktyp.
5. Redigera blockets fält i höger panel.
6. Klicka på **Spara**.

## Sortera block

Du kan dra blocken i vänster panel för att ändra ordning.

Ordningen i listan motsvarar ordningen i inlägget, uppifrån och ner.

## Åtgärder för block

Via blockets åtgärdsmeny kan du:

- kopiera blocket
- klistra in ett kopierat block
- duplicera blocket
- döpa om blocket
- dölja eller visa blocket
- ta bort blocket

Ett dolt block finns kvar i redigeraren men visas inte publikt.
