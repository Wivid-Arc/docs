Du kan flytta en bokning till ett annat kommande tillfälle för samma upplevelse.

## Omboka

1. Öppna bokningsdetaljen.
2. Välj åtgärden för att omboka.
3. Välj ett nytt tillfälle.
4. Välj om kunden ska få mejl.
5. Lägg till intern kommentar vid behov.
6. Spara.

## Vilka tillfällen kan väljas?

Listan visar kommande tillfällen för samma upplevelse.

Det aktuella tillfället visas inte som valbart.

## Mejl till kund

Om du väljer att skicka mejl får kunden information om den nya tiden.

<Note>
  Ombokning påverkar bokningens datum och tid, men inte nödvändigtvis kundens övriga uppgifter.
</Note>
