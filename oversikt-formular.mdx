Under **Formulär** hanterar du webbplatsens formulär.

## Öppna formulärlistan

1. Logga in i adminpanelen.
2. Välj rätt projekt.
3. Gå till **Webbplats**.
4. Öppna **Formulär**.

## Vad du ser i listan

Formulärlistan visar bland annat:

- formulärets namn
- mottagare för e-post
- antal inskick
- senaste inskick
- åtgärder för formuläret

Om ett formulär saknar e-postmottagare sparas inskick fortfarande i adminpanelen.

## Sök

Använd sökfältet för att hitta ett formulär via namn eller slug.

## Åtgärder

Via åtgärdsmenyn kan du:

- **Redigera** formuläret
- **Visa inskick**
- **Ta bort** formuläret
- återställa ett borttaget formulär
- radera ett borttaget formulär permanent

## När formulär används

Formulär kan visas på sidor eller i inlägg via formulärblock.
