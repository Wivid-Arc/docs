Du skapar nya inlägg från inläggslistan.

## Skapa ett nytt inlägg

1. Gå till **Webbplats**.
2. Öppna **Inlägg**.
3. Klicka på **Skapa inlägg**.
4. Fyll i inläggets uppgifter.
5. Välj om inlägget ska publiceras direkt.
6. Klicka på **Skapa**.

När inlägget är skapat visas det i inläggslistan och kan öppnas i redigeraren.

## Duplicera ett befintligt inlägg

Du kan också skapa ett nytt inlägg genom att duplicera ett inlägg som redan finns.

Det är användbart om du vill återanvända samma struktur, block, bildspråk eller upplägg.

1. Gå till **Webbplats**.
2. Öppna **Inlägg**.
3. Öppna åtgärdsmenyn för inlägget du vill utgå från.
4. Välj **Duplicera**.
5. Öppna den nya kopian och uppdatera innehållet.

<Warning>
  Kontrollera alltid kopian innan den publiceras. Byt namn, slug, SEO-titel, meta-beskrivning, utdrag, bild, taggar och inläggsinnehåll så att inlägget inte råkar publiceras med samma information som originalet.
</Warning>

## Fält

### Namn

Inläggets titel.

Titeln används i adminpanelen och kan även visas publikt beroende på inläggets layout.

### Slug

Sluggen är den sista delen av inläggets URL.

Exempel:

```text
min-forsta-post
```

Om arkivet har sluggen `/posts` blir inläggets URL:

```text
/posts/min-forsta-post
```

### Bild

Inläggets huvudbild.

Bilden används ofta i arkivet, i kort och vid delning beroende på webbplatsens inställningar.

### Taggar

Taggar hjälper till att kategorisera inlägg.

Om arkivet har filtrering aktiverad kan besökare filtrera inlägg efter taggar.

### Publiceringsstatus

Välj om inlägget ska vara:

- **Utkast**
- **Publicerad**

Du kan ändra status och schemalägga publicering senare i inläggets inställningar.

<Tip>
  Skapa gärna inlägg som utkast först om du vill bygga innehållet i lugn och ro innan publicering.
</Tip>
