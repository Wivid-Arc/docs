Sidans block är de innehållssektioner som bygger upp själva sidan.

Varje block har ett eget syfte, till exempel huvudbanner, text, bild, formulär, FAQ eller prisinformation. Du kombinerar block för att skapa sidans struktur.

## Lägga till block

1. Öppna sidan i redigeraren.
2. Gå till **Sidans block** i vänster panel.
3. Klicka på **Lägg till block**.
4. Välj blocktyp.
5. Redigera blockets fält i höger panel.
6. Klicka på **Spara**.

## Sortera block

Du kan ändra ordningen genom att dra blocken i listan.

Ordningen i vänster panel motsvarar ordningen på den publika sidan, uppifrån och ner.

## Åtgärder för ett block

Via blockets åtgärdsmeny kan du:

- **Kopiera** blocket
- **Klistra in** ett kopierat block
- **Duplicera** blocket
- **Dölj block** eller **Visa block**
- skapa eller koppla bort ett **globalt block**
- **Ta bort** blocket

<Tip>
  Använd **Duplicera** när du vill skapa flera liknande sektioner med samma struktur.
</Tip>

## Vanliga inställningar

Många block delar samma grundinställningar:

- **Titel** och **Text** styr blockets huvudbudskap.
- **Titelstorlek** och **Textstorlek** styr visuell nivå.
- **Titel som** styr rubriknivå, till exempel H1 eller H2.
- **Tema** kan vara system, ljust eller mörkt.
- **Vertikal padding** styr blockets höjd och luft.
- **Ram ovanför** och **Ram under** lägger till avgränsare.
- **Ankar-ID** gör det möjligt att länka direkt till blocket.
