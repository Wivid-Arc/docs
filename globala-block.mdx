Globala block är återanvändbara block som kan användas på flera sidor.

## Skapa ett globalt block

1. Öppna en sida i redigeraren.
2. Gå till **Sidans block** eller **Sidfot**.
3. Öppna blockets åtgärdsmeny.
4. Välj **Globalt block → Skapa**.
5. Ange ett namn.
6. Klicka på **Skapa**.

Blocket får då en global koppling och kan användas på andra sidor.

## Lägga till ett globalt block

1. Klicka på **Lägg till block**.
2. Gå till sektionen **Globala block** i blockmenyn.
3. Välj det globala block du vill lägga till.

Globala block visas tillsammans med vanliga block i vänster panel.

## Koppla bort ett globalt block

Om du vill göra blocket fristående:

1. Öppna blockets åtgärdsmeny.
2. Välj **Globalt block → Koppla bort**.

Efter det kan blocket redigeras utan att påverka andra sidor.

## När globala block passar

Använd globala block för innehåll som ska vara likadant på flera sidor, till exempel:

- återkommande CTA-sektioner
- kontaktsektioner
- kampanjblock
- informationsblock
- sidfotsblock

<Tip>
  Namnge globala block tydligt, till exempel `CTA - Boka demo` eller `Kontaktsektion - Standard`.
</Tip>

<Note>
  Ändringar i ett globalt block kan påverka alla sidor där blocket används.
</Note>
