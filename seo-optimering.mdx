SEO hjälper sökmotorer och besökare att förstå vad en sida handlar om.

I Polar CMS kan du hantera SEO-inställningar för varje sida och se en SEO-poäng som visar vad som kan förbättras.

## Öppna SEO-inställningar för en sida

1. Gå till **Webbplats**.
2. Öppna **Sidor**.
3. Öppna åtgärdsmenyn för sidan.
4. Välj **Inställningar**.
5. Gå till SEO-fälten i sidinställningarna.

Du kan också klicka på SEO-mätaren i sidlistan för att se vad som påverkar sidans poäng.

## Viktiga SEO-fält

I sidinställningarna kan du fylla i:

| Fält | Beskrivning |
| --- | --- |
| Titel för sökmotorer | Titeln som visas i sökresultat. |
| Beskrivning för sökmotorer | Kort text som sammanfattar sidan. |
| Sökord | Det viktigaste ordet eller frasen sidan ska fokusera på. |
| Open Graph-bild | Bild som används när sidan delas i sociala medier. |

## Titel för sökmotorer

Skriv en tydlig titel som beskriver sidans innehåll.

Bra riktlinjer:

- skriv sidans viktigaste ämne tidigt i titeln
- håll titeln tydlig och relevant
- undvik att alla sidor har samma titel

Exempel:

```text
Guidad stadsvandring i Stockholm
```

## Meta-beskrivning

Meta-beskrivningen är en kort sammanfattning av sidan.

Den bör förklara vad besökaren hittar på sidan och gärna uppmuntra till klick.

Exempel:

```text
Följ med på en guidad stadsvandring i Stockholm och upptäck stadens historia, gränder och berättelser.
```

## Sökord

Sökordet används för att bedöma hur väl sidan fokuserar på ett ämne.

Välj ett ord eller en fras som sidan faktiskt handlar om.

Exempel:

- `stadsvandring stockholm`
- `guidad tur gamla stan`
- `företagsevent stockholm`

<Tip>
  Välj hellre ett specifikt sökord än ett för brett ord. Det gör sidan lättare att optimera.
</Tip>

## Open Graph-bild

Open Graph-bilden används när sidan delas på sociala medier och i vissa meddelandeappar.

Välj en bild som representerar sidan och ser bra ut i breda format.

## SEO-poäng

I sidlistan visas en SEO-poäng för varje sida.

Klicka på poängen för att se:

- kritiska problem
- vilka kontroller som är godkända
- vilka delar som kan förbättras
- hur många poäng som saknas

SEO-poängen kan påverkas av bland annat:

- metadata
- rubrikstruktur
- innehåll
- media
- slug

## Publicera efter ändringar

När du har uppdaterat SEO-inställningarna behöver du spara ändringarna.

Kontrollera sedan gärna sidan i sidlistan för att se om SEO-poängen förbättrats.
